# Data Docs

This folder is the reference source for spreadsheet schema and dependencies.

## Recommended Read Order

1. `DatasheetGuide.md` for tab-by-tab sheet authoring and field intent.
2. `MinimalRuntimeSchema.md` for the smallest runnable schema subset.
3. `SpreadsheetSchema.md` for full table/field definitions.
4. `SchemaDependencyMap.md` for table relationships.
5. `SpreadsheetEnums.md` for controlled enum values.
