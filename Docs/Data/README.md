# Data Docs

This folder is the reference source for spreadsheet schema and dependencies.

## Recommended Read Order

1. `MinimalRuntimeSchema.md` for the smallest runnable schema subset.
2. `SpreadsheetSchema.md` for full table/field definitions.
3. `SchemaDependencyMap.md` for table relationships.
4. `SpreadsheetEnums.md` for controlled enum values.
